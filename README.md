# Portfolio Website

A one-page project portfolio built with plain HTML, CSS, and JavaScript.

## Edit your content

- Use `CONTENT-TODO.md` as the main checklist of placeholder text and links.
- Update your name, email, GitHub, and LinkedIn links in `index.html`.
- Replace `Project One`, `Project Two`, and `Project Three` with your real project names.
- Replace the placeholder images in `assets/projects/` with screenshots or mockups of your own work.
- Add your live project links and source code links in each `.project-links` block.
- Edit the tags in each `.tag-list` to show the tools you used.

## Open locally

Open `index.html` in your browser, or run a small local server from this folder:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.
